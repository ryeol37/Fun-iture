# Introduction
완제품 및 커스텀오더가 가능한 가구제작 쇼핑몰입니다.

전반적인 디자인과 기능들은 [MILLIENS](http://mosdesign.co.kr){:target="_blank"} 를 벤치마킹하여 제작하였습니다.

## 주요구현 기능
* 관리자
  * 회원 관리
  * 게시판 관리
  * 제품 관리
* 회원
  * 회원가입
  * 제픔 구매
  * 제품 커스텀 주문
  * 메일발송 시스템
  * etc...

## 나의 기여도
* 관리자
  * 상품 관리
  * 상품리뷰 관리
* 회원
  * 카테고리 분류
  * 제품 검색
  * 제품 상세보기
  * 제품 리뷰
  * 제품 추천
  * 제품 추천순 정렬
